Expense Reimbursement System (ERS)

(OK)-An Employee can login

(OK)-An Employee can view the Employee Homepage

(OK)-An Employee can logout

(OK)-An Employee can submit a reimbursement request

(OK)-An Employee can upload an image of his/her receipt as part of the reimbursement request

(OK)-An Employee can view their pending reimbursement requests

(OK)-An Employee can view their resolved reimbursement requests
(OK)-An Employee can view their information
(OK)-An Employee can update their information

    (optional)-An Employee receives an email when one of their reimbursement requests is resolved (optional)


(OK)-A Manager can login

(OK)-A Manager can view the Manager Homepage

(OK)-A Manager can logout

(OK)-A Manager can approve/deny pending reimbursement requests

(OK)-A Manager can view all pending requests from all Employees
(OK)-A Manager can view images of the receipts from reimbursement requests

(OK)-A Manager can view all resolved requests from all Employees and see which Manager resolved it

(OK)-A Manager can view all Employees

(OK)-A Manager can view reimbursement requests from a single Employee

------------------------------------------------------------------
-
A Manager can register an Employee, which sends the Employee an email with their username and temp password (optional)

-An Employee can reset their password (Optional - tied with above functional requirement)

-Servlets
-JDBC

-PL/SQL
-HTML5/CSS3

-Javascript/AJAX

-JUnit
